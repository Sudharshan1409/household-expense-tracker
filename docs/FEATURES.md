# Feature Catalogue

## Must-have
- Separate accounts
- Shared households
- Invitations
- Member roles
- Expense CRUD
- Payer vs beneficiary allocation
- Equal/percentage/fixed/custom splits
- Categories/subcategories
- Payment methods
- Month dashboard
- Member dashboard filtering
- Budgets
- Income/savings
- Search/filter
- Calendar
- Recurring transactions
- Settlements
- Reports
- Exports
- Responsive PWA
- Light/dark theme

## Dashboard analytics
- total
- actual member spend
- paid amount
- shared/personal
- savings/net cash flow
- budget
- period comparison
- daily average
- largest transaction
- highest-spend day
- monthly/daily/weekly trend
- category/payment/member distribution
- top expenses
- recent transactions

## Nice-to-have after core stability
- Receipt OCR
- Natural-language/voice entry
- Merchant auto-categorization
- Smarter duplicate detection
- Push reminders
- Vehicle/trip-specific views
- Credit-card statement tracking
- Assets/net worth

These must not delay the reliable core expense tracker.
